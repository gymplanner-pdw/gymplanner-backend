Sistema de Agendamento para Academia

Sobre o Projeto

Este é um sistema web desenvolvido para facilitar o agendamento de horários de uso de máquinas e equipamentos em academias. Ele tem como objetivo organizar e otimizar a utilização dos equipamentos, oferecendo uma experiência prática e eficiente para os usuários.

Funcionalidades

  • Cadastro de usuários com informações básicas.
  • Consulta e agendamento de horários para uso de equipamentos.

Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

  • Node.js e Express para o backend.
  • HTML, CSS e JavaScript para o frontend.
  • Banco de dados relacional para armazenamento de informações (PostgreeSQL).
